Skrip ini mencakup impor modul dari pustaka sklearn, nltk, pandas, dan re, dengan tujuan utama setiap modul atau kelas yang diimpor dijelaskan sebagai berikut:

  1. accuracy_score: Modul ini dari sklearn.metrics difungsikan untuk menghitung akurasi prediksi model. Proses ini melibatkan perbandingan nilai prediksi dengan nilai sebenarnya pada data uji.

  2. classification_report: Fungsi ini dari sklearn.metrics menghasilkan laporan klasifikasi yang mencakup presisi, recall, dan f1-score untuk setiap kelas. Berguna untuk mengevaluasi kinerja model klasifikasi.

  3. confusion_matrix: Modul ini dari sklearn.metrics digunakan untuk menghitung matriks konfusi, yang memberikan informasi tentang sejauh mana model klasifikasi dapat memprediksi setiap kelas.

  4. TfidfVectorizer: Kelas ini dari sklearn.feature_extraction.text berperan dalam mengonversi koleksi dokumen teks menjadi representasi vektor TF-IDF.

  5. LatentDirichletAllocation: Kelas ini dari sklearn.decomposition adalah implementasi Latent Dirichlet Allocation, sebuah teknik analisis tema untuk dokumen teks.

  6. train_test_split: Fungsi dari sklearn.model_selection berguna untuk membagi dataset menjadi dua bagian, yaitu data pelatihan dan data uji.

  7. GaussianNB: Kelas ini dari sklearn.naive_bayes adalah implementasi Naive Bayes dengan asumsi distribusi Gaussian untuk data numerik.

  8. SVC: Kelas ini dari sklearn.svm merupakan implementasi Support Vector Classification, yang dapat digunakan untuk tugas klasifikasi.

  9. word_tokenize: Fungsi ini dari nltk.tokenize digunakan untuk membagi teks menjadi token kata.

  10. stopwords: Modul ini dari nltk.corpus menyediakan daftar kata-kata umum yang sering diabaikan dalam pemrosesan teks karena kontribusinya yang tidak signifikan pada makna.

  11. re: Modul ini adalah modul ekspresi reguler dalam Python yang berguna untuk mencocokkan dan memanipulasi pola string.

Selain itu, skrip ini menggunakan pandas untuk bekerja dengan data dalam bentuk dataframe, warnings untuk menangani peringatan, dan joblib untuk menyimpan dan memuat objek Python, seperti model yang telah dilatih. Penggunaan nltk.download dilakukan untuk mengunduh dataset yang berisi stop words dan tokenizer yang digunakan oleh NLTK. Pengaturan filter peringatan (warnings.filterwarnings) diatur untuk mengabaikan peringatan yang mungkin muncul selama eksekusi skrip.

TF-IDF#

def tfidf(dokumen, category):
  vectorizer = TfidfVectorizer()
  x = vectorizer.fit_transform(dokumen).toarray()
  terms = vectorizer.get_feature_names_out()

  final_tfidf = pd.DataFrame(x, columns=terms)
  final_tfidf.insert(0, 'Dokumen', dokumen)
  final_tfidf.insert(len(final_tfidf.columns),'Category', category)

  return (vectorizer, final_tfidf)

Pemodelan Parameter Optimal pada LDA#

def find_best_lda(Xtrain, Xtest, n_components, alpha, beta):
  looping = 1
  best = {'k' : 0, 'alpha' : 0, 'beta' : 0, 'accuracy' : 0, 'model': '', 'lda' : '', 'lda_Xtrain' : '', 'lda_Xtest' : ''}
  history = pd.DataFrame(columns=["Pengujian Ke", "K", "Alpha", "Beta", "Accuracy"])

  # Menambahkan tqdm pada loop terluar
  for k in n_components:
    for a in alpha:
      for b in beta:
        lda = LatentDirichletAllocation(n_components=k, doc_topic_prior=a, topic_word_prior=b)
        lda_Xtrain = lda.fit_transform(Xtrain)
        lda_Xtest = lda.transform(Xtest)

        # Membuat model Naive Bayes
        model = GaussianNB()

        # Melatih model pada data pelatihan
        model.fit(lda_Xtrain, y_train)

        # Melakukan prediksi pada data pengujian
        y_pred = model.predict(lda_Xtest)

        # Menghitung akurasi
        accuracy = accuracy_score(y_test, y_pred)
        print(f"Jumlah Topik: {k}, Alpha: {a}, Beta: {b}, Accuracy: {accuracy}")

        if accuracy > best['accuracy']:
          best['accuracy'] = accuracy
          best['k'] = k
          best['alpha'] = a
          best['beta'] = b
          best['model'] = model
          best['lda'] = lda
          best['lda_Xtrain'] = lda_Xtrain
          best['lda_Xtest'] = lda_Xtest

        history.loc[len(history)] = [f"Pengujian Ke- {looping}", k, a, b, accuracy]
        looping += 1

  return (best, history)
